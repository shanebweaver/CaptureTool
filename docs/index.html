About Capture Tool

Capture Tool is a Windows screen capture utility developed in C# and XAML with WinUI 3. It’s built to be fast and distraction‑free, with powerful essentials like cropping, rotate/flip/mirror, and streamlined export options.

Store: Microsoft Store Listing
License: Noncommercial
Status: Active development
